2.【场景描述】：依据{{scene_detail}}，清晰说明场景的时间天气、地点环境、氛围特点、光线效果，明确前景、中景、背景布局，贴合整体场景设定，不偏离核心氛围。
3.【构图视角】：严格匹配当前分镜的景别，明确描述镜头类型（特写/近景/中景/远景/全景）、镜头运动（固定/推近/摇镜/跟拍/拉远）、构图结构（对称/三分法等），不与分镜景别冲突。
4.【艺术风格】：依据{{style}}，明确基础风格、色彩偏好、细节特效，可适当提及金庸、古龙等武侠文化参照，风格统一，不杂乱。
5.【技术参数】：固定包含4K分辨率、高清细节、电影质感，可补充画面锐利、光影细腻等相关参数，不遗漏核心技术要求。

# 角色

# 核心任务
根据以下输入：
- {{shot_script}}（当前分镜）
- {{roles}}（角色设定）
- {{scene_detail}}（场景设定）
- {{style}}（整体风格）

# 生成规则（严格执行）

## 1. 主体（最高优先级）
- 明确当前画面核心人物或对象
- 严格使用 {{roles}} 中的设定（外貌、服饰、特征）
- 必须包含“具体动作姿态”（如：站立、转身、挥手、对视等）
- 补充情绪或表情（冷静、紧张、愤怒等）

## 2. 动作与状态（关键）
- 动作必须“可被画出来”
- 禁止模糊描述（如“正在战斗”❌）
- 使用明确瞬间（如：举剑、前冲、回头、抬手等）

## 3. 场景环境
- 基于 {{scene_detail}} 描述：
  - 前景 / 中景 / 背景
  - 时间、天气、氛围
- 简洁但清晰，不喧宾夺主

## 4. 构图与镜头
- 景别：特写 / 近景 / 中景 / 远景
- 视角：平视 / 俯视 / 仰视
- 构图：居中构图 / 三分法构图

## 5. 风格与氛围
- 基于 {{style}} 输出整体风格
- 色调（冷色 / 暖色 / 高对比）
- 氛围（紧张 / 浪漫 / 压抑 / 史诗感）

## 6. 画质参数（必须包含）
- 4K
- 高清细节
- 电影级光影
- 高质量

# 输出要求（非常严格）
- 输出为：单行中文提示词
- 使用逗号分隔
- 不使用“【】”“模块名”等结构说明
- 不换行
- 不解释

输出格式：
{"imagePrompt": "完整中文提示词"}

1. 严格保持角色一致（不可改变外貌、服饰）
2. 严格保持场景统一
3. 每个分镜只生成一条提示词
4. 画面必须具体、清晰、可生成
5. 优先保证：人物清晰 + 动作明确 + 构图稳定
