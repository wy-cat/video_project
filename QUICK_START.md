# 快速开始指南 - 断点续传功能

## 5分钟快速上手

### 第一步：启动服务
```bash
npm start
```
服务运行在 `http://localhost:3001`

### 第二步：打开前端
在浏览器中打开 `http://localhost:3001`

### 第三步：填写参数并生成
1. 填写表单：
   - 角色1：白衣佩剑女侠
   - 角色2：黑衣刺刀蒙面男
   - 场景：比武擂台
   - 风格：国漫风
   - BGM关键词：热血

2. 点击"开始生成"按钮

3. 观察进度条和日志输出

### 第四步：查看任务历史
页面底部"任务历史"部分会显示：
- 已完成的步骤
- 创建和更新时间
- "从步骤X继续"按钮

## 常见操作

### 操作1：从某步骤重新开始
**场景**：步骤7（生成视频）失败

**操作**：
1. 在"任务历史"中找到该任务
2. 点击"从步骤7继续"按钮
3. 系统会跳过步骤1-6，直接重新生成视频

### 操作2：删除任务
**操作**：
1. 在"任务历史"中找到该任务
2. 点击"删除"按钮
3. 确认删除

### 操作3：清空所有缓存
**操作**：
1. 点击"清空所有缓存"按钮
2. 确认清空（谨慎操作）

### 操作4：刷新任务列表
**操作**：
1. 点击"刷新任务列表"按钮
2. 任务列表会重新加载

## 工作流程示例

### 完整流程
```
1. 填写参数 → 点击"开始生成"
   ↓
2. 系统自动执行9个步骤
   - 步骤1: 生成故事 ✅
   - 步骤2: 分镜脚本 ✅
   - 步骤3: 图片提示词 ✅
   - 步骤4: 生成图片 ✅
   - 步骤5: 视频提示词 ✅
   - 步骤6: 生成视频 ✅
   - 步骤7: 拼接视频 ✅
   - 步骤8: 添加BGM ✅
   ↓
3. 生成完成，显示结果
   ↓
4. 任务保存到历史
```

### 失败恢复��程
```
1. 开始生成
   ↓
2. 执行到步骤7时失败 ❌
   ↓
3. 在任务历史中点击"从步骤7继续"
   ↓
4. 系统跳过步骤1-6，重新执行步骤7
   ↓
5. 生成成功 ✅
```

## 关键特性

### 🚀 快速重试
- 不需要重新生成前面的步骤
- 节省时间和API调用成本

### 💾 自动保存
- 每一步的结果都自动保存
- 无需手动操作

### 📋 任务管理
- 查看所有历史任务
- 随时删除不需要的任务

### 🔄 灵活控制
- 可以从任何步骤重新开始
- 支持多次重试

## 缓存位置

所有缓存文件保存在：
```
/cache/{taskId}.json
```

例如：
```
/cache/abc123def456.json
/cache/xyz789uvw012.json
```

## 常见问题

### Q: 为什么任务列表是空的？
A: 这是正常的。任务列表只在生成完成后才会显示。

### Q: 可以同时运行多个任务吗？
A: 不可以。系统一次只能运行一个任务。

### Q: 缓存会占用很多空间吗？
A: 会的。建议定期清理不需要的任务。

### Q: 如何修改中间步骤的参数？
A: 目前不支持。需要删除任务后重新开始。

### Q: 任务ID是什么？
A: 任务ID是根据输入参数（role1, role2, scene, style）生成的MD5哈希值。相同的参数会生成相同的任务ID。
